# Augment

> 답이 아니라, 더 좋은 질문에 도달하기 위한 작업대.

더글러스 엥겔바트(1925–2013)의 *"Augmenting Human Intellect"* 철학을 따른 AI 사고 도구. 채팅이 아니라 **사고의 가지를 보존하는 트리**, 답변을 다른 시각으로 다시 보는 **ViewSpec**, 좋은 결론을 누적시키는 **Promote**.

## 핵심 원리

엥겔바트는 컴퓨터를 *답을 주는 기계*가 아니라 **인간이 더 어려운 질문을 다룰 수 있게 해주는 사고의 외골격**으로 보았습니다. 이 도구는 그 사상의 작은 구현입니다.

- **Automation** ❌ → **Augmentation** ⭕
- 선형 채팅 ❌ → 주소 가능한 노드 트리 ⭕
- 답변 중심 ❌ → 사고를 확장하는 도구 ⭕
- 휘발성 ❌ → 누적되는 지식 ⭕

## 기능

- **사고 트리** — 대화는 일자가 아니라 가지로 자랍니다. 모든 노드는 고유 주소(`#abc12345`)를 가집니다 (1968년 NLS의 *purple number*).
- **분기** — 답변의 어느 문장이든 클릭하면 그 지점에서 새 가지가 시작됩니다.
- **ViewSpec** — 같은 답을 5초 요약 / 3줄 핵심 / 반대 관점 / 다음 질문 등 다른 렌즈로 즉석 재렌더링.
- **Promote** — 좋은 답을 결정 기록으로 승격. 누적되는 살아있는 기억(Dynamic Knowledge Repository)의 씨앗.
- **메모 / 태그 / 핀** — 노드에 마진 메모, 태그, 핀을 달아 대화를 가로질러 연결.
- **노드 비교** — 갈라진 두 답의 공통점·차이·빠진 시각을 AI가 분석.
- **사고 그래프** — 대화 전체의 형태를 한눈에 조망.
- **Jump Palette (⌘K)** — 노드 ID·텍스트·메모·태그로 검색 점프.
- **인용 (#id)** — 입력창에 다른 노드를 끌어와 사고를 합칩니다.

## 시작하기

```bash
pnpm install
```

[.env.local](.env.local) 파일을 만들고 Anthropic API 키를 넣습니다:

```
ANTHROPIC_API_KEY=sk-ant-...
```

```bash
pnpm dev
```

[http://localhost:4020](http://localhost:4020) 에서 작동합니다.

## 기술

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Anthropic SDK (Claude Sonnet 4.6)
- 데이터: 로컬 JSON 파일 (`data/conversations/*.json`)

## 단축키

| 키 | 동작 |
|---|---|
| ⌘K | 점프 팔레트 열기 |
| ⌘⇧C | 두 노드 비교 |
| ⌘+Enter | 메시지 전송 |
| Esc | 모달·패널 닫기 |
| ⌘+B | 분기 (선택된 문장에서) |
